# Rigged Wheel
Wheel of Fortune with many options and customizations.

## Features:

### Basic Wheel of Fortune functionality:
- Add as many entries as you wish.
- Change the color of the entries.
- Customize spin time and winning text, more customization options coming soon.

### Secret Rigging Panel (open by clicking "by Ankanpää")
- Set hidden weights.
- Show fake weighted slices.
- Force winner and exclude from winning.
- Clear "unclearable" results history.
- Add fake result history
